## Description
Learn how to add production-ready request and response logging to a Django REST Framework API using DRF API Logger.

In this tutorial, I walk through why API logging matters, how to install and configure DRF API Logger, how request/response logs flow through the middleware, how to inspect logs in Django admin, and what to watch before enabling logging in production.

What you will learn:
- Install DRF API Logger in a Django REST Framework project
- Configure the app and middleware
- Store API logs in the database or consume logs with signals
- Inspect API requests, responses, status codes, client IPs, and execution time
- Mask sensitive fields before production
- Use profiling data to spot slow queries and N+1 patterns
